# Product screenshots

Drop a screenshot here for each product card and it appears automatically on the
homepage and `/work` — no code change needed. If a file is missing, the card
shows a neutral placeholder with the product name.

Use PNG, roughly 16:10 (e.g. 1600×1000), cropped to the product UI.

| Filename | Card |
|---|---|
| `alexai.png` | AlexAI (alexepstein.ai) |
| `alexai-pro.png` | AlexAI Pro (pro.alexepstein.ai) |
| `energenius.png` | Energenius |
| `ameshai.png` | AmeshAI (amesh.ai) |
| `ncea-ai.png` | NCEA AI |
| `blumbergai.png` | BlumbergAI |

The "Open source" cards (Open Debate, Argument Analyzer) don't use screenshots.

Do not add a screenshot that reveals the enterprise AlexAI Pro engagement.
